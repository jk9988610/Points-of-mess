# 在 Pad 上怎么用这个项目？

> **安卓平板用户请直接看：[安卓平板-看这里.md](安卓平板-看这里.md)**（Termux 安装、在哪输入命令、Chrome 打开网址，逐步说明）

## 先记住一件事

**文件夹里有很多文件，但浏览器里只需要打开一个网址。**

| 以前（单 HTML） | 现在（多文件） |
|----------------|----------------|
| 在「文件」里点 `xxx.html` 就能看 | **不要**在「文件」里点 `index.html`（会白屏或无法对话） |
| 密钥可能写在网页里 | 密钥在 `.env`，由 `server.js` 代理，更安全 |

多出来的 `js/`、`styles/` 等是给**编辑器**用的，浏览器会自动加载，你不用逐个打开。

---

## 三步开始（在 Pad 本机开发 / 测试）

### 1. 把整个项目文件夹放到 Pad

例如用 **Working Copy**、**文件**、**iCloud 云盘** 同步整个 `Points-of-mess` 文件夹（不是只拷一个 html）。

### 2. 在终端里启动（只需做一次安装）

在 Pad 上打开能跑命令的 App，进入项目目录后执行：

```bash
sh pad-start.sh
```

**常用终端 App：**

- **iPad**：iSH（App Store 搜 iSH Shell）、a-Shell
- **Android 平板**：Termux

首次会 `npm install`，较慢属正常。若提示缺少 `.env`，用文本编辑器打开 `.env`，填入：

```env
DEEPSEEK_API_KEY=你的密钥
```

再执行一次 `sh pad-start.sh`。

### 3. 用浏览器打开终端里打印的地址

- **安卓平板**：Chrome，地址 `http://127.0.0.1:3000`
- **iPad**：Safari，地址 `http://127.0.0.1:3000` 或 `http://localhost:3000`

**把这个地址加到主屏幕**，以后测试只点这一个，不用管有多少个文件。

---

## 改代码时改哪些？

| 想改… | 编辑这个文件 |
|--------|----------------|
| 页面文字、按钮 | `index.html` |
| 颜色、布局 | `styles/chat.css` |
| 发送、停止、清空逻辑 | `js/app.js` |
| 对话记录保存 | `js/state.js` |
| API 密钥、端口 | `.env`（不要写进 html） |

改完后：终端里 `Ctrl+C` 停掉服务，再 `sh pad-start.sh`；若开着 `npm run dev` 则保存后刷新 Safari 即可。

---

## 还是打不开？

1. 确认终端里 **没有报错**，且出现 `Chat server listening`。
2. 地址必须是 **`http://`** 开头，不是 `file://`。
3. 详细说明见 [docs/PAD-开发指南.md](docs/PAD-开发指南.md)。
