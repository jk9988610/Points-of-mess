# 计划：地图桌面文档（Plan 模式）

## 1. Context

用户希望像 **电脑桌面** 一样，在地图上打开 Plan / 说明类文档，而不是只能读仓库里的 md 文件。

## 2. Goals

| # | 目标 |
|---|------|
| G1 | 地图上可见「文档图标」，靠近后可点击打开 |
| G2 | 弹层内阅读 `docs/*.md`（含 PLAN-api-reliability） |
| G3 | 与「锋利」对话互不挡路；对话中可关闭文档 |

## 3. Non-goals

- 不做完整 Markdown 渲染器（先纯文本 + 简单标题行）。
- 不在文档里再嵌 AI。

## 4. 方案

- `js/desktop.js`：文档清单、命中检测、fetch、弹层。
- `map.js`：绘制图标（色块 + 标题）。
- `app.js`：点击优先级：文档 > 角色 > 移动。

## 5. 验收

- [ ] 靠近「PLAN·API」可打开 PLAN-api-reliability.md
- [ ] 靠近「常用说明」可打开 常用说明.md
- [ ] 关闭按钮 / 点遮罩可关
